### Requirements

1. C++ compiler
2. CMake & Make

### How to start?

1. Clone the repo

```sh
git clone https://github.com/budchirp/calcplusplus && cd calcplusplus
```

2. Compile the code

```sh
mkdir build && cd build
cmake .. && make -j$(nproc)
```

> NOTE: If you're using fish, remove the `$` sign.

3. Have fun!

```sh
./calcplusplus
```
